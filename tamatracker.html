TamaTracker

Gamifying kendama progress tracking

  • UI Design

Background

TamaTracker is a web application that helps kendama players learn new tricks and track their progress. I designed, developed, and tested a basic version of TamaTracker in 2021. The project collected dust until 2024, when I redesigned the tool based on user feedback gathered years prior.
